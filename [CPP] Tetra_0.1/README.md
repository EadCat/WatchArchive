# EadCat Tetra 0.1
## Created on Sept. 2019. Windows VS

There are many bugs.  
Build Windows only.

Guide  
1. No Block clear method. (just stacking)   
2. No Gravity method.  
3. 'A' button : left rotation  
4. 'S' button : right rotation  
5. 'space bar' button : down the block  
6. 'Arrow keys' (방향키) : move the block
